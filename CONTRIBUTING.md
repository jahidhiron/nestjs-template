# Contributing to NestJS Template

We welcome contributions to this project! To get started, please follow these guidelines to help us maintain a smooth workflow.

## Reporting Issues

If you find a bug or have suggestions for improvements, please create a new issue in the "Issues" section of the repository. When reporting an issue, include the following:

- A clear description of the problem.
- Steps to reproduce the issue.
- Any relevant logs or error messages.
- Your development environment (e.g., Node.js version, Docker setup, etc.).

## Submitting Pull Requests

We encourage you to submit pull requests for improvements, bug fixes, and new features. Please follow these steps:

1. **Fork the repository**:
   - Click the "Fork" button on the top-right of the repository page to create your own copy.

2. **Clone your fork** to your local machine:
   ```bash
   git clone git@github.com:your-username/nestjs-template.git
   cd nestjs-template
   ```
